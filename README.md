## Это тестовая задача по разработке страницы с онлайн-курсами. 

Проект включает:

1. Первую секцию с заголовком и бейджем.
2. Фильтры с кнопками категорий и поиском.
3. Список карточек курсов с изображениями, категориями, названиями, ценами и авторами.
4. Кнопку "Load more" для загрузки дополнительных карточек.
5. Декоративные элементы как фон под основным контентом.

## Структура проекта
1. index.html: Основная разметка страницы.
2. styles/global.css: Глобальные стили, переменные и ресет.
3. styles/style.css: Основные стили для компонентов.
4. assets/: Изображения, шрифты и декор.
5. js/...: Скриптs для динамики (фильтры, загрузка карточек, кнопок).
6. data/courses.json: Джсон файл с данными для карточек

## Технологии
1. HTML5: Семантическая разметка.
2. CSS: Cтили и переменные.
3. JavaScript: Для взаимодействия.

В проекте не используются никакие библиотеки.
